# Webnovel Writer 安装指示

请执行以下步骤完成 Webnovel Writer 安装：

## 步骤 1：下载安装脚本

从 https://raw.githubusercontent.com/lujih/webnovel-writer-opencode/master/install.py 下载内容，保存为 install.py

## 步骤 2：运行安装

执行 python install.py

脚本会自动：
- 检查并更新到最新版本
- 清理旧版本 .opencode 目录
- 下载并配置所有文件

## 完成

确认安装成功：
- .opencode 目录已创建
- .env 文件存在（保留原有配置）
