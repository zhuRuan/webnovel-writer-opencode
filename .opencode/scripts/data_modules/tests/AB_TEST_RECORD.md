# 分词增强效果 A/B 盲测记录表

## 测试说明

本记录表用于验证 **Day 1-2 分词增强** 改进的效果。

### 测试方法

1. **旧版（单字符分词）**：原始 `_tokenize` 方法，中文按字符切分
2. **新版（jieba + 数字归一化）**：
   - 使用 jieba 智能分词
   - 数字归一化（`3年` → `三年`）
   - 自定义词典（网文专有词汇）

- **10分**: 完全准确，结果精确匹配
- **8分**: 基本准确，有少量无关结果
- **6分**: 部分准确，需要筛选
- **4分**: 勉强相关
- **2分**: 基本不相关
- **0分**: 无结果
