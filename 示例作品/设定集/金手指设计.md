# 金手指设计

## 设定定位
- 题材适配点：末世生存 + 古法修行，硬核克制风格
- 主要作用：推进主线（修炼体系）+ 解决硬约束（预知代价）+ 放大爽点（信息差、成长爽）
- 读者可见度：只有主角知道（陈默对他人隐瞒）

## 常见类型参考
- 系统 / 空间 / 位面穿梭 / 重生记忆 / 传承 / 血脉 / 召唤

## 类型
传承型 + 老爷爷型（书灵/守护者）

## 核心功能
- 功能描述：《洞天要术》是上古修炼文明的传承载体，滴血认主后激活识海空间，书灵守护者提供修炼指导
- 可视化表现：识海中出现古书光影，可翻阅书页获取信息；修炼时有灵气流转感
- 触发条件：滴血认主（Chapter 3）

## 获得方式
- 触发事件：暑假工第一天，在地摊花10块钱购得
- 代价/限制：需要每日修炼维持
- 不可逆代价：每一次预知未来，都会磨损一丝人性

## 使用规则
- 冷却/次数限制：预知能力每7天可用一次；修炼无限制但进度取决于主角悟性和资源
- 禁止事项：
  - 禁止向他人透露金手指真实性质
  - 禁止过度依赖预知能力
- 失败惩罚：预知失败会加速人性磨损
- 反制方式（敌人如何克制）：隔绝神识的禁制可屏蔽书灵（如地下遗迹石室）

## 升级路线
- 初始形态：感气境（Chapter 3-20）
- 中期提升：引气境（Chapter 21-40）
- 最终形态：筑基→开光→融合→辟谷→金丹→元婴→化神

## 爽点嵌入
- 获得爽：地摊捡漏、滴血认主成功、识海初开
- 成长爽：境界突破、体质增强、感知范围扩大
- 使用爽：预知未来获取信息差、古书揭示上古秘密

## 反馈节奏
- 关键反馈节点（每N章必须体现）：每10章至少一次书灵互动
- 代价兑现节点：Chapter 5首次预知人性磨损
- 反转节点：Chapter 31揭示书灵身份线索

## 与创意约束对齐
- 反套路规则如何体现：预知能力有代价（人性磨损），而非无代价的系统面板
- 硬约束如何绑定金手指：
  - 约束1（不透露金手指）：通过剧情强制执行
  - 约束2（人性磨损）：与预知能力绑定
  - 约束3（修炼速度慢）：古法修炼需要资源积累
- 主角缺陷如何被放大/修复：
  - 过度谨慎 → 书灵提供信息帮助决策，但会消耗人性代价
  - 冷眼旁观 → 修炼提升实力，但人性磨损让他更难共情
