# 力量体系设定

## 体系公理
- 核心信条/定律：力量越大，代价越大；修炼之路孤独而漫长
- 代价规则：每次使用预知能力，都会永久失去一部分人性
- 公平性原则（谁能破格、谁必须付出代价）：古法修炼者不能依赖金手指快速变强，必须付出时间与努力

## 体系类型
- 体系类型：三足鼎立（古法修炼、异能觉醒、科技残存）
- 典型境界链（可选）：感气 → 引气 → 筑基 → 开光 → 融合 → 辟谷 → 金丹 → 元婴 → 化神
- 小境界划分：每个境界可拆3-4个小阶段（如筑基前期/中期/后期/大圆满）

### 常见境界链参考（可选）
- 修真链：练气 → 筑基 → 金丹 → 元婴 → （后续自定义）

## 能力来源
- 能量/资源来源：灵气复苏（全球范围）、晶核（变异生物体内）、古法传承（《洞天要术》）
- 训练/修炼方法：古法修炼（内功心法、灵气引导）、异能觉醒（基因突变、晶核吸收）、科技残存（热武器、机械改造）
- 社会控制机制：无统一组织，陈默独自修炼；觉醒者由势力控制；科技残存由旧军方控制

## 等级体系
- 等级顺序：感气（感知灵气）→ 引气（引导灵气入体）→ 筑基（开辟丹田）→ 开光（灵气外放）→ 融合（灵气与血肉融合）→ 辟谷（脱离食物依赖）→ 金丹（丹田凝实体）→ 元婴（能量分身）→ 化神（肉身与灵气完美融合）
- 每级核心能力：
  - 感气：体质渐强，感知灵气存在
  - 引气：力气变大，可引导灵气入体
  - 筑基：容貌锁定，可辟谷三日
  - 开光：灵气外放，附着器物
  - 融合：自愈力大增
  - 辟谷：完全脱离食物依赖
  - 金丹：可短途飞行
  - 元婴：神识覆盖百里
  - 化神：肉身与灵气完美融合

## 晋级条件
- 资源要求：灵气浓度、晶核辅助、天材地宝
- 突破方式：积累灵气、突破瓶颈、心境感悟
- 失败代价：走火入魔、修为倒退、人性丧失
- 越级代价（越阶战斗/突破）：反噬重伤、根基受损、寿命缩短

## 资源系统
- 核心资源类型：灵气、晶核、天材地宝、古法传承
- 资源获取方式：修炼吸收、猎杀变异生物、探索遗迹、交易获取
- 稀缺性规则：灵气分布不均、晶核等级越高越稀有、天材地宝可遇不可求
- 替代路径/灰色交易：黑市交易、势力掠夺、遗迹探险

## 战斗规则
- 伤害与防御逻辑：灵气护体、武器附灵、异能对抗、科技武器
- 战斗节奏特点：古法修炼持久战、异能觉醒爆发战、科技残存远程战
- 克制/反制关系：古法修炼克制异能觉醒（根基扎实）、异能觉醒克制科技残存（灵活多变）、科技残存克制古法修炼（远程压制）
- 逃生/撤退机制：灵气遁术、异能瞬移、科技烟雾弹

## 禁忌与限制
- 禁术/禁地：预知能力不可滥用、古法传承不可外泄
- 高阶限制：化神期以上需渡劫、元婴期以上神识消耗巨大
- 硬限制（不可绕过）：
  1. 主角不能向他人透露金手指的真实性质
  2. 每次使用预知能力，都会永久失去一部分人性
  3. 主角的修炼速度必须比觉醒者慢，不能依赖金手指快速变强

## 体系漏洞与利用
- 漏洞（必须可解释）：古法修炼速度慢但根基扎实、异能觉醒速度快但上限模糊、科技残存威力大但资源有限
- 主角如何利用：利用古法修炼的根基优势，在长期战斗中获胜；利用预知能力规避风险
- 反派如何反制：掠夺者靠数量压制、觉醒者靠爆发力、科技残存靠远程火力

## 反馈节奏
- 境界变化节点：每卷至少1次小境界突破、每2卷至少1次大境界突破
- 读者可感知的战力差距展示方式：战斗场面描写、境界威压对比、技能效果展示

## 上古修炼体系（第49章揭示）

### 灵气自我循环系统
- **核心原理**：上古修炼者不依赖外界灵气，体内灵气可在灵气节点间循环往复、自我增殖
- **灵气节点**：上古修炼者体内有数十个灵气节点（不同于现代穴位），灵气在节点间循环
- **与现代体系对比**：
  - 现代古法修炼：从外界吸收灵气 → 体内积累 → 突破境界（线性增长）
  - 上古修炼体系：体内灵气自我循环 → 自我增殖（指数增长，但需要"渊"作为源头）
- **致命缺陷**：该体系依赖"渊"作为力量源，当"渊"产生"归墟"意志时，整个体系崩溃

### 万灵归墟
- **本质**：不是修炼终局，而是一种意志——要将万灵归于虚无的意志
- **触发条件**：当"渊"产生"归墟"欲望时触发
- **后果**：上古文明因此覆灭（具体过程被刻意抹去）
- **与陈默的关系**：陈默的修炼之路最终也要面对"万灵归墟"
